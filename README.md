# 解开心结训练营 - 东方奢雅·水墨禅意版

> 120分钟内感受明显变化，21天专业陪伴的心理疗愈服务

## 项目概述

本项目是一个具有东方禅意美学的专业心理健康服务宣传页面，采用"水墨禅意"设计风格，融合传统文化元素与现代专业标准，为用户提供温暖、专业、可信赖的心理疗愈服务体验。

### 核心特色

- **东方奢雅设计**：水墨禅意美学，传递文化内涵与专业形象
- **专业级标准**：WCAG AAA无障碍标准，黄金比例视觉层级
- **响应式设计**：完美适配桌面端、平板端、移动端
- **文化融合**：中文数字、禅宗留白、水墨动画等传统元素

## 设计理念

### 东方奢雅·水墨禅意 (Eastern Luxury · Ink Wash Zen)

融合禅宗"留白"、"意境"与现代奢侈品设计"克制"、"精致"的设计哲学，创造出既具有传统文化底蕴又符合现代审美的视觉体验。

### 色彩系统

| 色彩名称 | 色值 | 应用场景 |

| 竹月青 | #8FBC8F | 行动按钮、核心标识 |

### 设计亮点

- **留白美学**：超大间距（64-128px）营造呼吸感
- **书法感字体**：思源宋体传递文化底蕴
- **淡墨晕染**：抽象水墨动画象征"解开心结"过程
- **中文数字**：用"一二三四五"替代阿拉伯数字

## 部署说明

### GitHub Pages 部署

1. **上传到GitHub仓库**：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <your-repo-url>
   git push -u origin main
   ```

2. **启用GitHub Pages**：
   - 进入仓库 Settings → Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main" 
   - Folder 选择 "/ (root)"
   - 点击 Save

3. **访问页面**：
   - 访问 `https://<username>.github.io/<repository-name>`
   - 页面将自动显示 index.html

### 本地预览

```bash
# 使用Python简单服务器
python -m http.server 8000

# 或使用Node.js serve
npx serve .

# 访问 http://localhost:8000
```
